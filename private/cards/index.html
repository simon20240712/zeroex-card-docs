    卡片与交易 — /private¶

    面向持卡用户，提供卡片操作、交易查询、3DS 验证等功能。

    卡片列表¶

    获取当前用户持有的全部卡片（ACTIVE / FROZEN / SUSPENDED 状态）。

    请求

    GET /private/cards

    响应示例

    {
  "code": 0,
  "msg": "success",
  "data": [
    {
      "cardId": "xxx",
      "cardNoLast4": "1234",
      "email": "user@example.com",
      "phoneCode": "1",
      "phoneCountryCode": "US",
      "phone": "xxx",
      "username": "John",
      "status": "Active",
      "virtual": true,
      "available": 100.00,
      "country": "US",
      "state": "CA",
      "city": "SF",
      "street": "xxx",
      "address": "xxx",
      "postcode": "94000",
      "cardImage": "https://...",
      "forwardType": "API_SMS",
      "cardImageId": 1,
      "cardImageUrl": "https://...",
      "cardImageThumbUrl": "https://...",
      "expYear": "28",
      "expMonth": "12",
      "recentUsed": false,
      "label": "我的卡"
    }
  ]
}

    字段说明（CardPrivateInfo）

    : cardId {#field-cardid} : string — 卡片唯一标识

    : cardNo {#field-cardno} : string — 完整卡号（仅安全验证后返回）

    : cardNoLast4 : string — 卡号后 4 位

    : cardAddress : string — 卡片账单地址

    : email : string — 邮箱

    : phoneCountryCode : string — 手机国家代码（2 位，如 US）

    : phoneCode : string — 手机区号（如 1）

    : phone : string — 手机号

    : cvv : string — CVV 安全码（仅安全验证后返回）

    : expYear : string — 有效期年份（两位，如 28）

    : expMonth : string — 有效期月份（两位，如 12）

    : username : string — 卡片昵称

    : status : string — 状态：Active / Frozen / Suspended

    : virtual : boolean — 是否虚拟卡

    : available : decimal — 可用余额（USD）

    : recentUsed : boolean — 是否为最近使用的卡

    : country : string — 国家

    : state : string — 州或省

    : city : string — 城市

    : street : string — 街道

    : address : string — 街道地址

    : postcode : string — 邮政编码

    : cardImage : string — 卡片展示图片 URL

    : forwardType : string — 3DS 通知方式：API_SMS / API_EMAIL / NO

    : cardImageId : long — 仓库卡面 ID

    : cardImageUrl : string — 正面图 URL 快照

    : cardImageThumbUrl : string — 缩略图 URL 快照

    卡片详情¶

    获取指定卡片的详细信息，支持安全验证后返回完整卡号和 CVV。

    请求

    POST /private/card
Content-Type: application/json

    请求体（CardDetailForm）

    参数 类型 必填 说明
    cardId string ✅ 卡片 ID
    token string — 安全令牌；传入则验证后返回完整卡号和 CVV

    说明

    • 不传 token：返回基本信息（卡号、CVV 字段为 null）
    • 传入 token 且验证通过：额外返回 cardNo（完整卡号）和 cvv

    token 有效期

    安全令牌由前端通过短信/人脸等验证流程获取，有效期 5 分钟。

    冻结 / 解冻卡片¶

    冻结或解冻卡片资金，冻结后卡片不可消费。

    请求

    POST /private/frozen
Content-Type: application/json

    请求体（CardFrozenForm）

    参数 类型 必填 说明
    cardId string ✅ 卡片 ID
    token string ✅ 安全令牌
    frozen boolean ✅ true = 冻结，false = 解冻
    remark string — 冻结原因

    响应

    { "code": 0, "msg": "success", "data": null }

    删除卡片¶

    删除指定卡片，系统自动将余额转出到现货钱包后再注销卡片。

    请求

    POST /private/delete
Content-Type: application/json

    请求体（CardDeleteForm）

    参数 类型 必填 说明
    cardId string ✅ 卡片 ID
    token string ✅ 安全令牌

    响应

    { "code": 0, "msg": "success", "data": null }

    说明

    • 删除前自动扣收当月月费
    • 余额转出到现货钱包后再注销
    • 卡片状态变为 Deleted

    限额调整¶

    用户修改自己卡片的消费限额（需后台开启 allowUserModifyLimit 权限）。

    请求

    POST /private/velocity_control
Content-Type: application/json

    请求体（UserLimitForm）

    参数 类型 必填 说明
    cardId string — 卡片 ID
    token string — 安全令牌（3.8.7 前版本不要求）
    monthlyTotal decimal ✅ 月累计限额
    dailyTotal decimal ✅ 日累计限额
    perTransaction decimal ✅ 单笔限额
    remark string — 备注

    用户自行修改条件

    仅当 CardUser.allowUserModifyLimit = true 时用户才可自行调整限额。

    修改 3DS 通知方式¶

    切换 API_SMS / API_EMAIL / NO 等 3DS 验证通知方式。

    请求

    POST /private/update_forward
Content-Type: application/json

    请求体（CardUpdateForwardForm）

    参数 类型 必填 说明
    cardId string ✅ 卡片 ID
    forwardType string ✅ 通知方式
    token string ✅ 安全令牌

    forwardType 可选值

    值 说明
    API_SMS API 短信通知
    API_EMAIL API 邮件通知
    NO 不通知，跳转页面验证

    交易记录与 3DS 验证¶

    交易记录列表¶

    支持普通分页和游标分页两种模式。

    请求

    GET /private/transactions

    参数

    参数 类型 必填 说明
    cardId string — 筛选指定卡片
    type string — 交易类型（单个）
    types list — 交易类型（批量）
    currency string — 结算币种
    transactionCurrency string — 原始交易币种
    status string — 交易状态
    startTime long — 交易时间开始（毫秒时间戳）
    endTime long — 交易时间结束（毫秒时间戳）
    orderType string — UI 层交易类型筛选
    orderStatus string — UI 层状态筛选
    id long — 精确查询指定交易 ID
    relatedId long — 精确查询关联 ID（如退款查原消费）
    lastId long — 游标分页：上一页最后一条 ID
    page int — 普通分页页码，默认 1
    rows int — 每页条数，默认 10

    分页逻辑

    • 传 id 或 relatedId（精确查询）：使用普通分页
    • 传 lastId：使用游标分页，返回 id < lastId 的记录
    • 其他情况：默认游标分页，查��最近 365 天数据

    交易类型（type）

    值 说明
    CONSUME 消费（授权）
    REFUND 退款（清算后）
    CONSUME_REFUND 消费退款（已清算）
    MONTH_FEE 月费扣收
    TRANSFER_IN 充值（划转入）
    TRANSFER_OUT 提现（划转出）
    FLASH_IN 闪兑入

    交易状态（status）

    值 说明
    PENDING 处理中
    AUTHORIZED 已授权（等待清算）
    CLEARED 已清算
    DECLINED 已拒绝
    FAILED 失败

    响应示例

    {
  "code": 0,
  "msg": "success",
  "data": {
    "count": 100,
    "pageNum": 1,
    "pageSize": 10,
    "list": [
      {
        "id": 123,
        "uid": 10001,
        "cardId": "xxx",
        "cardNo": "1234",
        "cardName": "John",
        "currency": "USD",
        "amount": "-50.0000",
        "transactionCurrency": "USD",
        "transactionAmount": "-50.0000",
        "fee": "0.5000",
        "exchangeRate": 1,
        "type": "CONSUME",
        "status": "CLEARED",
        "transactionTime": "2025-01-01T12:00:00",
        "virtual": true,
        "remark": null,
        "mccCode": "5411",
        "mccCategory": "Grocery Stores",
        "merchant": "AMAZON",
        "merchantCountry": "US",
        "relatedId": null
      }
    ]
  }
}

    字段说明（CardTransactionVO）

    : id {#field-id} : long — 交易记录 ID

    : uid : long — 用户 ID

    : cardId : string — 卡片 ID

    : cardNo : string — 卡号（通常为后 4 位）

    : username : string — 用户名

    : cardName : string — 卡片姓名（username 或 label）

    : currency : string — 结算币种（USD）

    : amount : string — 订单数量（负数 = 支出，正数 = 收入）

    : transactionCurrency : string — 交易币种

    : transactionAmount : string — 对方收到数量

    : fee : string — 交易费用

    : exchangeRate : decimal — 汇率

    : relatedId : long — 关联交易 ID（退款时关联原消费）

    : type : string — 订单类型（TransactionTypeEnum）

    : status : string — 交易状态（TransactionStatusEnum）

    : transactionTime : datetime — 交易时间

    : virtual : boolean — 是否虚拟卡

    : remark : string — 备注说明

    : mccCode : string — 商户类别码

    : mccCategory : string — 商户类别名称

    : merchant : string — 对方名称（商户）

    : merchantCountry : string — 对方所在国家

    金额字段为字符串

    amount、transactionAmount、fee 使用字符串返回，避免浮点数精度丢失。

    3DS 验证¶

    用户在收到 3DS 推送后，提交验证码或授权结果。

    请求

    POST /private/answer
Content-Type: application/json

    请求体（Card3dsAnswerForm）

    参数 类型 必填 说明
    txId long ✅ 交易 ID
    approve boolean ✅ 用户是否授权（true = 同意，false = 拒绝）
    cardId string — 卡片 ID
    merchantName string — 商户名称
    merchantCountryCode string — 商户国家代码
    transactionAmount decimal — 交易金额
    transactionCurrency string — 交易币种
    transactionTimestamp long — 交易时间戳

    响应

    { "code": 0, "msg": "success", "data": null }
