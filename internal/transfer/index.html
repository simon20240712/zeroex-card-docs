    资产划转 — /internal/transfer*¶

    现货钱包与卡片钱包之间的双向资金划转。

    支持的货币¶

    请求

    POST /internal/support_currencies?brand={brand}
    参数 类型 必填 说明
    brand string ✅ 品牌

    响��

    {
  "code": 0,
  "msg": "success",
  "data": ["USDT"]
}

    闪兑检查¶

    检查用户是否满足闪兑条件（卡是否存在、风控、币种支持）。

    请求

    POST /internal/flash_check
Content-Type: application/json

    请求体（CardFlashForm）

    参数 类型 必填 说明
    uid long ✅ 用户 UID
    targetId string ✅ 目标卡片 ID
    sourceCurrency string ✅ 源币种，如 ETH
    brand string ✅ 品牌
    language string — 语言代码

    响应（成功）

    {
  "code": 0,
  "msg": "success",
  "data": {
    "checkResult": true,
    "code": null,
    "message": null
  }
}

    响应（失败）

    {
  "code": 0,
  "msg": "success",
  "data": {
    "checkResult": false,
    "code": 10001,
    "message": "卡片不存在"
  }
}

    CheckResultVO 字段说明

    : checkResult : boolean — 是否通过

    : code : int — 错误码（失败时非空）

    : message : string — 错误信息（失败时非空）

    修改卡片预留信息¶

    请求

    POST /internal/update_user_card_info
Content-Type: application/json

    请求体（UpdateUserCardProfileParam）

    | 参数 | 类型 | 必填 | 说明 | |---|:---:|:---||---| | uid | long | ✅ | 用户 UID | | brand | string | ✅ | 品牌 | | operator | string | ✅ | 操作人 | | cardId | string | ✅ | 卡片 ID | | ... | ... | ... | 其他字段见实体定义 |

    响应

    { "code": 0, "msg": "success", "data": true/false }

    卡片资产余额¶

    查询用户在指定品牌下的全部卡片资产余额（USDT 折算）。

    请求

    POST /internal/support_assets?uid={uid}&cardId={cardId}&brand={brand}
    参数 类型 必填 说明
    uid long ✅ 用户 UID
    cardId string — 卡片 ID（不传则返回全部卡）
    brand string ✅ 品牌

    响应

    {
  "code": 0,
  "msg": "success",
  "data": [
    {
      "uid": 10001,
      "currency": "USDT",
      "balance": 500.00,
      "brand": "BYD"
    }
  ]
}

    CardAssetVO 字段说明

    : uid : long — 用户 UID

    : currency : string — 币种（USD 卡余额折算为 USDT）

    : balance : decimal — 可用余额

    : brand : string — 品牌

    资产划转¶

    现货钱包 ↔ 卡片钱包之间的资金划转（核心接口）。

    请求

    POST /internal/transfer

    Header

    Header 类型 必填 说明
    nonce string ✅ 时间戳
    sign string ✅ 签名

    请求体（TransferRequest）

    参数 类型 必填 说明
    uid long ✅ 用户 UID
    txid string ✅ 外部事务 ID（幂等键）
    source string ✅ 源钱包：SPOT / LITE / SWAP / AGENT / UCENTER
    target string ✅ 目标钱包：SPOT / CARD
    sourceId string — 源钱包 ID（划出时用）
    targetId string — 目标钱包 ID（划入时用）
    currency string ✅ 币种（目前仅支持 USDT）
    amount string ✅ 划转金额
    brand string ✅ 品牌
    params map — 闪兑参数：sourceCurrency / sourceAmount / convertPrice

    划转类型判断

    source target 类型
    SPOT CARD 转入（充值）
    CARD SPOT 转出（提现）
    其他组合 抛出异常

    手续费

    方向 费率
    转入（CARD） 1%（进卡金额 = 发起金额 × (1 - 1%)）
    转出（SPOT） 1%（到账金额 = 发起金额 × (1 - 1%)）

    响应

    code 说明
    0 成功
    100 划转处理中（异步）
    200 划转成功
    500 失败（参数错误或系统异常）
    600 余额不足（仅转出时）
    601 体验金钱包不支持划转
    602 不支持的划转路径
    591 划转记录不存在
    592 划转已失败
    593 划转已退款
    598 划转失败，请联系客服
    599 重复划转（相同 txid）

    响应示例

    { "code": 200, "msg": "Transfer record existed, result is success", "data": "txid_xxx" }

    划转退款¶

    对已成功的转出划转（SPOT ← CARD）进行退款。

    请求

    POST /internal/transfer/refund

    Header

    Header 类型 必填 说明
    nonce string ✅ 时间戳
    sign string ✅ 签名

    请求体（TransferRefundRequest）

    | 参数 | 类型 | 必填 | 说明 | |---|:|---||---| | txid | string | ✅ | 原划转 txid | | brand | string | ✅ | 品牌 |

    响应

    使用相同的状态码体系，见 资产划转。

    划转查询¶

    请求

    POST /internal/transfer/query?txid={txid}&brand={brand}

    | 参数 | 类型 | 必填 | 说明 | |---|:|:|:---||---| | txid | string | ✅ | 划转 txid | | brand | string | ✅ | 品牌 |

    响应

    code 说明
    100 处理中
    200 成功
    591 划转记录不存在
    592 划转失败
    593 划转已退款
    594 系统错误（非法状态）
